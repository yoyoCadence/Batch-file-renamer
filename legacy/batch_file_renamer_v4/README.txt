Batch File Renamer v4
=====================

No Python required. No install required.

Run:
- Double-click Run_BatchFileRenamer_v4.cmd

Changes:
1. Template file selection uses a dedicated single-file picker.
2. Large preview editor is non-modal, so the main window remains editable.
3. Irrelevant fields are disabled/greyed out based on Value Mode.
4. Value List mismatch shows warning and keeps existing preview.
5. Light/Dark theme switching.
6. Added Value Mode = Delete.
7. Added live highlight example for Segment / Character target.
8. Segment supports FromStart / FromEnd.
9. Static value preserves spaces and does not trim automatically.
10. Added Import CSV.
11. Main preview grid expands when the window is resized.

CSV columns:
Action, SourcePath, TargetName, TargetFolder, TargetPath

Only rows with Status = OK are executed.
